# Shipping Quote Program
**Purpose: This assignment explores the use of parallel arrays and range matching.**<br>
<p align="justify">You are to create a calculator that calculates the price of a final shipping contract for a business’s products.
Each of the four shipping companies has a percentage discount rate (see Table 1) for six business partners listed in Table 2.
The user also needs to input the length of the contract to calculate an additional discount/saving depicted in Table 3. The longer the contract, the larger the discount. For example, if the user enters 6 for Contract Years, then an Additional Discount is $10,000.</p>
